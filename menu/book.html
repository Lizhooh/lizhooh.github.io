Koa2 学习笔记


Koa2 Learning Notes



作者：Lizhooh

类别：软件开发/后端技术

时间：2017 年 6 月









《 Koa2 学习笔记 》是我学习 Koa2 中记录的学习笔记，以 Express 的开发经验，来快速学习 Koa2。

这并不是入门级的书籍，而是需要你具有一定的 Nodejs 基础，ES6 & 7 语法基础，后端开发基础等，才能领会其中的奥义。里面覆盖了常见的 Nodejs Web 开发解决方案。





React 服务端渲染实战


React Server Render In Action



作者：Lizhooh

类别：软件开发/前端技术

时间：2018 年 5 月









在本书里，你将了解到 React 服务端渲染的基础知识、前端与 Nodejs 技术融合以及一些 React 服务端渲染框架的使用，还将学习到了如何对现有的服务端渲染应用进行性能优化，不同方式实现的服务端渲染应用示例。

本书并不会介绍太多的基础知识点，在阅读本书前，希望你了解并熟悉 ES6、React、Koa 等基本知识，并且具有 Node.js 相当不错的使用经验。
