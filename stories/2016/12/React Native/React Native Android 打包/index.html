React Native Android 打包需要注意的几点：

Android应用在发布前，需要生成发布版本安装包，然后上传至发布平台
React Native Android 打包分为以下几步：

  1. 生成发布密钥签名
  2. 修改 gradle 配置文件
  3. 生成发布版本安装包
