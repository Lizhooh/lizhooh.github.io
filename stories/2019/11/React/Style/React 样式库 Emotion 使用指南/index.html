前言

Emotion 是一个比 styled-components 更加轻量级的 CSS in JS 样式库。

除了提供出色的开发人员体验（如源映射，标签和测试实用程序）外，它还提供了强大且可预测的样式组成。字符串和对象样式均受支持。
