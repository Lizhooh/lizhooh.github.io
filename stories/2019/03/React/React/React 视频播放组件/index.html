前言

Griffith 是知乎前端团队开源的一个 React HTML5 视频播放组件，目前已在知乎 web 和 mobile web 内使用，并在 GitHub 上开源。

Griffith 具有的特性：

  • 快捷键支持 - Griffith 参考 YouTube 进行了快捷键支持
  • 状态管理 - Griffith 使用 Context API 进行状态管理。对于 React 应用，可以通过引入 Griffith 的 Context 来实现弹幕等自定义功能。
  • 丰富的事件系统 - 对于视频播放器中常见的首帧时长，缓冲次数等指标，可以通过接收 Griffith 事件来进行打点记录。
  • 支持流式播放 - Griffith 使用了 Media Source Extensions™ ，支持对 mp4 和 m3u8 格式的视频进行流式播放。
