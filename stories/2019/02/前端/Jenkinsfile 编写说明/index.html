前言

Jenkinsfile 是 jenkins 执行任务的描述文件。在大多时候会在项目里创建一个 Jenkinsfile 文件，让一些项目管理工具（coding、github 等）可以根据 Jenkinsfile 进行持续部署。

基本语法

一些注意事项

  • 不要在 jenkins 里执行一些不合结束的命令，这样会导致整个持续部署永无终止。
  • 有些需要权限的命令记得开启权限。
