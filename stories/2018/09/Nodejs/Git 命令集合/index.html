Git

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

几个专门用的名词：

  • Workspace：工作区
  • Index / Stage：暂存区
  • Repository：仓库区（或本地仓库）
  • Remote：远程仓库

工作流

你的本地仓库由git维护的三棵树组成。
第一个是 你的工作目录（Workspace），它持有实际文件；
第二个是 缓存区（Index），它像个缓存区域，临时保存你的改动；
第三个是 HEAD，指向你最近一次提交后的结果。
