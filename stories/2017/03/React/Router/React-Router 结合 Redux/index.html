前言

React单独使用，并不难。
React 与 React-Router 结合使用，可能会有点难度。
React 与 React-Redux 结合使用，可能会有点难度。

但它们三者都结合在一起React, React-Router, Redux使用的时候，会感觉让人眼花缭乱。

(这里的 React-Router 指的是 4.0 版本)
