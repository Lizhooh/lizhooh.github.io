修饰符

修饰符部分相当于 SQL 中的 Where 谓词，部分是 MongoDB 特有的。

Fields 字段

$inc 相当于 +=
$mul 相当于 *=

$rename 更改字段名称

$setOnInsert 当 update 方法使用 upsert 选项执行 insert 操作时，$setOnInsert 操作符给相应的字段赋值
若 update 方法执行的 update 操作而不是 insert 操作，那么 $setOnInsert 操作符将无效

$set 设置文档中一个字段的值

$unset 从文档中删除指定的字段(属性)

在 MongoDB 里，数据的存储格式是 BSON 文档格式，类似 JSON。并不像 SQL 中的表那样要求数据字段一致。

$min 如果指定的值小于现有的字段值则更新字段的值
$max 如果指定的值大于现有的字段值则更新字段的值

$currentDate 将一个字段设置为日期类型，默认为 date 类型，可选择：date,timestamp

Array 数组

$ 作为一个占位符来更新第一个元素相匹配的字段

$addToSet 若字段为数组，向字段尾追加新值，除非它已经存在。如果字段不是数组，则操作将失败。

$pop 若字段为数组，则向数组删除数据

$pullAll 从现有的所有实例数组删除指定的值

$pull 删除所有数组元素匹配指定的查询。

$push 若字段为数组，则向数组增加数据

Modiflers

$each 配合$push使用，遍历进行多个$push操作

$slice 能够限制数组的长度，相当于队列，超出的元素，将会把前面的挤出去。正数尾出，负数头出

$sort 对数组进行排序, -1降序, 1升序

$position 用来指定 $push 所插入的位置
