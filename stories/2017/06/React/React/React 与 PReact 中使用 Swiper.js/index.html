Swiper

关于 Swiper 的介绍：

  • Swiper 是一个非常强大的视图滑动（幻灯片，轮播图）库，BAT 各大公司都在使用。
  • Swiper 是纯 javascript 打造的滑动特效插件，面向手机、平板电脑等移动终端。
  • Swiper 能实现触屏焦点图、触屏 Tab 切换、触屏多图切换等常用效果。
  • Swiper 开源、免费、稳定、使用简单、功能强大，是架构移动终端网站的重要选择！
