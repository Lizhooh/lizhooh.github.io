MongoDB

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。其次 MongoDB 是一个 NOSQL，与 Mysql 这些 SQL 数据库有着不一样的地方。

数据储存差异

MongoDB 与 Mysql 的存储方式结构是不同的。Mysql 数据存储以表为单元，即一个二维结构，而 MongoDB 的数据存储以文档为单元，即一个多维结构。

因为 Mysql 在查询时需要作 SQL 语法树的解析，如果 SQL 非常复杂，在 SQL 解析上也是需要一定的时间的。
