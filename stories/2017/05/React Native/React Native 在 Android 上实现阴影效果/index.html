Shadow

Shadow阴影效果，在 ios 上是完全支持的，但在 Android 上只有 5.0 以上的系统支持类似的阴影效果（用elevation来实现)。
无赖，看了一下别人的react-native-shadow封装库，是用SVG实现，坑爹的是在新版本上运行不起来（作者好久没更新了，issue 又没反应）。

奈何，我不会 SVG，还好会 CSS3，后来还是用 CSS3 搞定了。
