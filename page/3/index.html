    1. 用户按发语音按钮，设置长按或者单按也行。
    2. 调用录音 API，开始录音，当然记得添加获取权限功能。
    3. 用户结束录音，接着把音频上传到后端，后端返回资源的 url，上传前客户端可以先处理一下。
    4. 拿到音频资源的 url，添加到消息体的结构里，发送出去，完成。
