    React Native 动画 Animated

    Animated

    在 CSS3 中创建动画使用：animation，而在 React Native 中创建动画则是使用 Animated。

    使用 Animated 需要三个步骤：

    1. 使用 new Animated.value(0); 初始化，动画值。
    2. 使用 <Animated.View> 包裹动画视图，并设置它的 style。
    3. 使用 Animated.timing(attr, { … }).start()等; 设置动画参数，并启动。
